# Templatecop

[RuboCop](https://github.com/rubocop/rubocop) runner framework for template language.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'templatecop'
```

And then execute:

```
bundle install
```

Or install it yourself as:

```
gem install templatecop
```
